# Women-Safety-Alert-System

**Women Safety Alert Prototype using Arduino**

**📌 Project Overview**

The Women Safety Alert Prototype is an Arduino-based embedded system designed to provide a quick emergency alert mechanism. When the SOS button is pressed, the system activates a buzzer alarm, turns on an LED indicator, displays an emergency message on the LCD screen, and records the alert count.

This project was developed and simulated using Tinkercad to demonstrate the basics of embedded systems, alert mechanisms, and user safety solutions.

**🚀 Features**
SOS Emergency Button
Reset Button
LED Emergency Indicator
Buzzer Alarm System
16x2 LCD Status Display
Alert Counter
Tinkercad Simulation

**🛠️ Components Used**
Arduino Uno
16x2 LCD Display
Push Button (SOS)
Push Button (Reset)
Piezo Buzzer
LED
220Ω Resistor
Potentiometer
Breadboard
Jumper Wires
